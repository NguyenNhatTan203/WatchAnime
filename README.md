## 🚀 Tính Năng

- ✅ Xem anime trực tuyến chất lượng cao  
- ✅ Tìm kiếm anime theo tên  
- ✅ Phân loại theo thể loại (Action, Romance, Comedy, …)  
- ✅ Giao diện thân thiện, tối ưu trên **mobile** và **desktop**  
- ✅ Dễ dàng mở rộng, cập nhật anime mới

```

```
```
cd WatchAnime
```
✅ 2. Chạy project
```
npm install
```
```
 npm start
```
